# Welcome to the Assemblyline User Manual

This documentation is designed to help you get started with using Assemblyline 4, as well as provide detailed information on its features and capabilities.

"The Basics" section will cover the fundamental concepts and workflows of Assemblyline. This includes how to submit files for analysis, how to interpret the results, and how to search for specific information within the system.

The "Advanced Features" section will delve into more complex functionalities of Assemblyline and new features that have been added.
