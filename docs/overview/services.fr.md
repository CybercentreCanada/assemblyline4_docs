# Services d'Assemblyline

Les Services installé sur un système peuvent être trouvé sous: `Help > Service Listing`.

Cette liste contient tous les services inclus et maintenue avec Assemblyline:

| Service Name                                                                                           | Speciality                 | Description                                                                                                                                                                                                              |
| ------------------------------------------------------------------------------------------------------ | -------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| [APIVector](https://github.com/CybercentreCanada/assemblyline-service-apivector)                       | Windows binaries           | Extrait les imports de fichiers PE files ou mémoire pour générer un vecteur de classification.                                                                                                                           |
| [APKaye](https://github.com/CybercentreCanada/assemblyline-service-apkaye)                             | Android APK                | Les APK sont décompilés et inspectés. Les indicateurs de réseau et les informations trouvées dans le fichier manifeste APK sont affichés                                                                                 |
| [AntiVirus](https://github.com/CybercentreCanada/assemblyline-service-antivirus)                       | Anti-virus                 | Client ICAP générique utilisant plusieurs solution d'Anti-virus commerciales                                                                                                                                             |
| [Batchdeobfuscator](https://github.com/CybercentreCanada/assemblyline-service-batchdeobfuscator)       | Deobfuscation              | Résolution de variables locales et globales d'un fichier batch                                                                                                                                                           |
| [CAPA](https://github.com/CybercentreCanada/assemblyline-service-capa)                                 | Windows binaries           | Intégration de l'outil public [CAPA](https://github.com/mandiant/capa)                                                                                                                                                   |
| [Characterize](https://github.com/CybercentreCanada/assemblyline-service-characterize)                 | Analyze d'entropy          | Calcule l'entropy des fichiers et extrait les meta-donnée Exif.                                                                                                                                                          |
| [ConfigExtractor](https://github.com/CybercentreCanada/assemblyline-service-configextractor)           | Extraction                 | Extrait la configuration de malware connu, pour trouvé des liste de C2s, cle d'encryption etc.                                                                                                                           |
| [CAPE](https://github.com/CybercentreCanada/assemblyline-service-cape)                                 | Sandbox                    | Fournit une analyse dynamique des logiciels malveillants via le sandboxing.                                                                                                                                              |
| [DeobfuScripter](https://github.com/CybercentreCanada/assemblyline-service-deobfuscripter)             | Deobfuscation              | Déobfuscation de script statique.                                                                                                                                                                                        |
| [ELF](https://github.com/CybercentreCanada/assemblyline-service-elf)                                   | Linux binaries             | Analyze de fichier ELF (sections, segments, ...) avec [Python library LIEF](https://github.com/lief-project/LIEF)                                                                                                        |
| [ELFPARSER](https://github.com/CybercentreCanada/assemblyline-service-elfparser)                       | Linux binaries             | Intégration de l'outil public [ELFParser](https://github.com/jacob-baines/elfparser)                                                                                                                                     |
| [EmlParser](https://github.com/CybercentreCanada/assemblyline-service-emlparser)                       | Email                      | Analyze de fichier email avec [GOVCERT-LU eml_parser library](https://github.com/GOVCERT-LU/eml_parser) comme les attachements, URL etc                                                                                  |
| [Espresso](https://github.com/CybercentreCanada/assemblyline-service-espresso)                         | Java                       | Extrait les classes Java, décompile et analyze pour comportement malicieux                                                                                                                                               |
| [Extract](https://github.com/CybercentreCanada/assemblyline-service-extract)                           | Compressed file            | Extrait la plus part des type de compression (like ZIP, RAR, 7z, ...)                                                                                                                                                    |
| [Floss](https://github.com/CybercentreCanada/assemblyline-service-floss)                               | IoC extraction             | Extrait des chaîne de charracters obfusqué avec [FireEye Labs Obfuscated String Solver](https://github.com/fireeye/flare-floss)                                                                                          |
| [FrankenStrings](https://github.com/CybercentreCanada/assemblyline-service-frankenstrings)             | IoC extraction             | Ce service effectue des extractions de fichiers et d'IOC à l'aide de la correspondance de modèles, d'un décodeur d'encodage simple et de désobfuscateurs de script                                                       |
| [Intezer](https://github.com/CybercentreCanada/assemblyline-service-intezer)                           | File genome identification | Interface entre Intezer Analyze API 2.0, soumet le fichier pour analyse si le hachage n'est pas présent dans la base de données Intezer                                                                                  |
| [IPArse](https://github.com/CybercentreCanada/assemblyline-service-iparse)                             | Apple IOS                  | Analyze de fichier Apple IOS                                                                                                                                                                                             |
| [JsJaws](https://github.com/CybercentreCanada/assemblyline-service-jsjaws)                             | Javascript                 | Analyze de fichier Javascript                                                                                                                                                                                            |
| [MetaPeek](https://github.com/CybercentreCanada/assemblyline-service-metapeek)                         | Meta data analysis         | Détect les signe malicieux dans les meta-données et les noms de fichier (double extension etc)                                                                                                                           |
| [Oletools](https://github.com/CybercentreCanada/assemblyline-service-oletools)                         | Office documents           | Ce service analyze les fichiers Office et extrait des indicateurs de compromis avec [Python library py-oletools](https://github.com/decalage2/oletools) by [Philippe Lagadec](https://linktr.ee/decalage)                |
| [Overpower](https://github.com/CybercentreCanada/assemblyline-service-overpower)                       | PowerShell                 | Déobfusque les fichier powershell                                                                                                                                                                                        |
| [PDFId](https://github.com/CybercentreCanada/assemblyline-service-pdfid)                               | PDF                        | Analyze de fichier PDF avec Didier Stevens [PDFId](https://github.com/DidierStevens/DidierStevensSuite/blob/master/pdfid.py) & [PDFParse](https://github.com/DidierStevens/DidierStevensSuite/blob/master/pdf-parser.py) |
| [PE](https://github.com/CybercentreCanada/assemblyline-service-pe)                                     | Windows binaries           | Analyze de fichier Windows (imports, exports, sections, ...) avec [LIEF](https://github.com/lief-project/LIEF)                                                                                                           |
| [PeePDF](https://github.com/CybercentreCanada/assemblyline-service-peepdf)                             | PDF                        | Ce service utilise [Python PeePDF library](https://github.com/jesparza/peepdf) pour extraire de l'information de fichier PDF                                                                                             |
| [PixAxe](https://github.com/CybercentreCanada/assemblyline-service-pixaxe)                             | Images                     | Extrait du text des images                                                                                                                                                                                               |
| [Safelist](https://github.com/CybercentreCanada/assemblyline-service-safelist)                         | Safelisting                | Permet de "safelister" des indicateur dans le système comme des domaines, hash etc [NSRL](https://www.nist.gov/itl/ssd/software-quality-group/national-software-reference-library-nsrl)                                  |
| [Sigma](https://github.com/CybercentreCanada/assemblyline-service-sigma)                               | Eventlog signatures        | Analyze de "Windows Event logs" avec les règles [Sigma](https://github.com/SigmaHQ/sigma)                                                                                                                                |
| [Suricata](https://github.com/CybercentreCanada/assemblyline-service-suricata)                         | Network signatures         | Analyze les captures réseaux (pcap) avec [Suricata](https://github.com/OISF/suricata)                                                                                                                                    |
| [Swiffer](https://github.com/CybercentreCanada/assemblyline-service-swiffer)                           | Adobe Shockwave            | Analyze de fichier Shockwave (.swf)                                                                                                                                                                                      |
| [TagCheck](https://github.com/CybercentreCanada/assemblyline-service-yara)                             | Tag signatures             | Signatures YARA sur les tags d'Assemblyline Tags                                                                                                                                                                         |
| [TorrentSlicer](https://github.com/CybercentreCanada/assemblyline-service-torrentslicer)               | Torrent files              | Analyze de fichier torrent                                                                                                                                                                                               |
| [Unpacker](https://github.com/CybercentreCanada/assemblyline-service-unpacker)                         | UPX Unpacker               | Extrait des executable a partir de fichier "packer" avec UPX                                                                                                                                                             |
| [Unpac.me](https://github.com/CybercentreCanada/assemblyline-service-unpacme)                          | Unpacker                   | Intégration avec [unpac.me](https://www.unpac.me)                                                                                                                                                                        |
| [URLCreator](https://github.com/CybercentreCanada/assemblyline-service-urlcreator)                     | Création de fichiers d'URL | Crée des fichiers URI à partir des tags URI apparemment malveillantes                                                                                                                                                    |
| [URLDownloader](https://github.com/CybercentreCanada/assemblyline-service-urldownloader)               | URL Fetching               | Récupère les URL des fichiers URI                                                                                                                                                                                        |
| [ViperMonkey](https://github.com/CybercentreCanada/assemblyline-service-vipermonkey)                   | Office documents           | [ViperMonkey](https://github.com/decalage2/ViperMonkey) est un programme d'emulation pour VBA par <https://linktr.ee/decalage>                                                                                           |
| [VirusTotal](https://github.com/CybercentreCanada/assemblyline-service-virustotal)                     | Anti-virus                 | Ce service vérifie (et soumet éventuellement) les fichiers/URL à VirusTotal pour analyse.                                                                                                                                |
| [XLMMacroDeobfuscator](https://github.com/CybercentreCanada/assemblyline-service-XLMMacroDeobfuscator) | Office documents           | Analyse de Macro Excel 4.0                                                                                                                                                                                               |
| [YARA](https://github.com/CybercentreCanada/assemblyline-service-yara)                                 | File signatures            | Signatures de fichier                                                                                                                                                                                                    |
