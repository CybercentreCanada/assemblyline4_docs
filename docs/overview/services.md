# Assemblyline services

Services currently installed on a system can be found under `Help > Service Listing`.

This is the list of all the services that are bundled with Assemblyline and that are maintained by the Assemblyline team:

| Service Name                                                                                           | Speciality                 | Description                                                                                                                                                                                                                                                                                                                                                            |
| ------------------------------------------------------------------------------------------------------ | -------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [APIVector](https://github.com/CybercentreCanada/assemblyline-service-apivector)                       | Windows binaries           | Extracts library imports from windows PE files or memory dump to generate api vector classification.                                                                                                                                                                                                                                                                   |
| [APKaye](https://github.com/CybercentreCanada/assemblyline-service-apkaye)                             | Android APK                | APKs are decompiled and inspected. Network indicators and information found in the APK manifest file are displayed                                                                                                                                                                                                                                                     |
| [AntiVirus](https://github.com/CybercentreCanada/assemblyline-service-antivirus)                       | Anti-virus                 | Generic ICAP client to integrate with most Anti-virus enterprise scanners                                                                                                                                                                                                                                                                                              |
| [Batchdeobfuscator](https://github.com/CybercentreCanada/assemblyline-service-batchdeobfuscator)       | Deobfuscation              | Deobfuscate batch file through variable resolution                                                                                                                                                                                                                                                                                                                     |
| [CAPA](https://github.com/CybercentreCanada/assemblyline-service-capa)                                 | Windows binaries           | [CAPA](https://github.com/mandiant/capa) open-source tool integration                                                                                                                                                                                                                                                                                                  |
| [Characterize](https://github.com/CybercentreCanada/assemblyline-service-characterize)                 | Entropy analysis           | Partitions the file and calculates visual entropy for each partition, extract Exif metadata                                                                                                                                                                                                                                                                            |
| [ConfigExtractor](https://github.com/CybercentreCanada/assemblyline-service-configextractor)           | IoC extraction             | Extract malware configuration file, allowing to get list of C2, encryption material etc.                                                                                                                                                                                                                                                                               |
| [CAPE](https://github.com/CybercentreCanada/assemblyline-service-cape)                                 | Sandbox                    | Provides dynamic malware analysis through sandboxing.                                                                                                                                                                                                                                                                                                                  |
| [DeobfuScripter](https://github.com/CybercentreCanada/assemblyline-service-deobfuscripter)             | Deobfuscation              | Static script de-obfuscator. The purpose is not to get surgical de-obfuscation, but rather to extract obfuscated IOCs.                                                                                                                                                                                                                                                 |
| [ELF](https://github.com/CybercentreCanada/assemblyline-service-elf)                                   | Linux binaries             | Extracts attributes (sections, segments, ...) from ELF files using [LIEF](https://github.com/lief-project/LIEF)                                                                                                                                                                                                                                                        |
| [ELFPARSER](https://github.com/CybercentreCanada/assemblyline-service-elfparser)                       | Linux binaries             | [ELFParser](https://github.com/jacob-baines/elfparser) open-source tool integration                                                                                                                                                                                                                                                                                    |
| [EmlParser](https://github.com/CybercentreCanada/assemblyline-service-emlparser)                       | Email                      | Parse emails using [GOVCERT-LU eml_parser library](https://github.com/GOVCERT-LU/eml_parser) while extracting header information, attachments, URIs                                                                                                                                                                                                                    |
| [Espresso](https://github.com/CybercentreCanada/assemblyline-service-espresso)                         | Java                       | All classes are extracted, decompiled, and analyzed for malicious behaviour                                                                                                                                                                                                                                                                                            |
| [Extract](https://github.com/CybercentreCanada/assemblyline-service-extract)                           | Compressed file            | This service extracts embedded files from file containers (like ZIP, RAR, 7z, ...)                                                                                                                                                                                                                                                                                     |
| [Floss](https://github.com/CybercentreCanada/assemblyline-service-floss)                               | IoC extraction             | Automatically extract obfuscated strings from malware using [FireEye Labs Obfuscated String Solver](https://github.com/fireeye/flare-floss)                                                                                                                                                                                                                            |
| [FrankenStrings](https://github.com/CybercentreCanada/assemblyline-service-frankenstrings)             | IoC extraction             | This service performs file and IOC extractions using pattern matching, simple encoding decoder and script de-obfuscators                                                                                                                                                                                                                                               |
| [Intezer](https://github.com/CybercentreCanada/assemblyline-service-intezer)                           | File genome identification | Interface between Intezer Analyze API 2.0, submits file for analysis if hash is not present in Intezer database                                                                                                                                                                                                                                                        |
| [IPArse](https://github.com/CybercentreCanada/assemblyline-service-iparse)                             | Apple IOS                  | Analyze Apple apps                                                                                                                                                                                                                                                                                                                                                     |
| [JsJaws](https://github.com/CybercentreCanada/assemblyline-service-jsjaws)                             | Javascript                 | Analyze malicious Javascript                                                                                                                                                                                                                                                                                                                                           |
| [MetaPeek](https://github.com/CybercentreCanada/assemblyline-service-metapeek)                         | Meta data analysis         | Checks submission metadata for indicators of potential malicious behaviour (double file extensions, ...)                                                                                                                                                                                                                                                               |
| [Oletools](https://github.com/CybercentreCanada/assemblyline-service-oletools)                         | Office documents           | This service extracts metadata, network information and reports anomalies in Microsoft OLE and XML documents using the [Python library py-oletools](https://github.com/decalage2/oletools) by [Philippe Lagadec](https://linktr.ee/decalage)                                                                                                                           |
| [Overpower](https://github.com/CybercentreCanada/assemblyline-service-overpower)                       | PowerShell                 | De-obfuscate PowerShell scripts                                                                                                                                                                                                                                                                                                                                        |
| [PDFId](https://github.com/CybercentreCanada/assemblyline-service-pdfid)                               | PDF                        | This service extracts metadata from PDFs using Didier Stevens [PDFId](https://github.com/DidierStevens/DidierStevensSuite/blob/master/pdfid.py) & [PDFParse](https://github.com/DidierStevens/DidierStevensSuite/blob/master/pdf-parser.py)                                                                                                                            |
| [PE](https://github.com/CybercentreCanada/assemblyline-service-pe)                                     | Windows binaries           | Extract attributes (imports, exports, sections, ...) from PE files using [LIEF](https://github.com/lief-project/LIEF)                                                                                                                                                                                                                                                  |
| [PeePDF](https://github.com/CybercentreCanada/assemblyline-service-peepdf)                             | PDF                        | This service uses the [Python PeePDF library](https://github.com/jesparza/peepdf) information from PDFs including JavaScript blocks which it will attempt to de-obfuscate, if necessary, for further analysis                                                                                                                                                          |
| [PixAxe](https://github.com/CybercentreCanada/assemblyline-service-pixaxe)                             | Images                     | Extract text from images                                                                                                                                                                                                                                                                                                                                               |
| [Safelist](https://github.com/CybercentreCanada/assemblyline-service-safelist)                         | Safelisting                | Allow for hash, IoC and signature safelisting, including support for downloading [NSRL](https://www.google.com/url?sa=t&rct=j&q=&esrc=s&source=web&cd=&cad=rja&uact=8&ved=2ahUKEwjaptDzi-7yAhWEdN8KHefxC6oQFnoECAsQAw&url=https%3A%2F%2Fwww.nist.gov%2Fitl%2Fssd%2Fsoftware-quality-group%2Fnational-software-reference-library-nsrl&usg=AOvVaw3I05XBysnEGMtwgozhlm8g) |
| [Sigma](https://github.com/CybercentreCanada/assemblyline-service-sigma)                               | Eventlog signatures        | Scan event logs (e.g. from sandbox or a compromised host) using [Sigma](https://github.com/SigmaHQ/sigma)                                                                                                                                                                                                                                                              |
| [Suricata](https://github.com/CybercentreCanada/assemblyline-service-suricata)                         | Network signatures         | Scan network capture (.pcap) submitted and extracted from analysis via [Suricata](https://github.com/OISF/suricata)                                                                                                                                                                                                                                                    |
| [Swiffer](https://github.com/CybercentreCanada/assemblyline-service-swiffer)                           | Adobe Shockwave            | This service extracts metadata and performs anomaly detection on Adobe Shockwave (.swf) files                                                                                                                                                                                                                                                                          |
| [TagCheck](https://github.com/CybercentreCanada/assemblyline-service-yara)                             | Tag signatures             | YARA signatures on Assemblyline Tags (build your own signatures to hit on specific tags)                                                                                                                                                                                                                                                                               |
| [TorrentSlicer](https://github.com/CybercentreCanada/assemblyline-service-torrentslicer)               | Torrent files              | Extracts information from torrent files                                                                                                                                                                                                                                                                                                                                |
| [Unpacker](https://github.com/CybercentreCanada/assemblyline-service-unpacker)                         | UPX Unpacker               | This service unpacks UPX packed executables for further analysis                                                                                                                                                                                                                                                                                                       |
| [Unpac.me](https://github.com/CybercentreCanada/assemblyline-service-unpacme)                          | Unpacker                   | Integrate with [unpac.me](https://www.unpac.me)                                                                                                                                                                                                                                                                                                                        |
| [URLCreator](https://github.com/CybercentreCanada/assemblyline-service-urlcreator)                     | URL File creation          | Create URI files from URLs tags that are seemingly malicious                                                                                                                                                                                                                                                                                                           |
| [URLDownloader](https://github.com/CybercentreCanada/assemblyline-service-urldownloader)               | URL Fetching               | Fetches URLs from URI files                                                                                                                                                                                                                                                                                                                                            |
| [ViperMonkey](https://github.com/CybercentreCanada/assemblyline-service-vipermonkey)                   | Office documents           | [ViperMonkey](https://github.com/decalage2/ViperMonkey) is a VBA Emulation engine by <https://linktr.ee/decalage>                                                                                                                                                                                                                                                      |
| [VirusTotal](https://github.com/CybercentreCanada/assemblyline-service-virustotal)                     | Anti-virus                 | This service checks (and optionally submits) files/URLs to VirusTotal for analysis.                                                                                                                                                                                                                                                                                    |
| [XLMMacroDeobfuscator](https://github.com/CybercentreCanada/assemblyline-service-XLMMacroDeobfuscator) | Office documents           | Analyze Excel 4.0 macros                                                                                                                                                                                                                                                                                                                                               |
| [YARA](https://github.com/CybercentreCanada/assemblyline-service-yara)                                 | File signatures            | Signature for file                                                                                                                                                                                                                                                                                                                                                     |
